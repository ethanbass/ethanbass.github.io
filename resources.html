Resources

Some software tools I developed during my PhD to facilitate easier and more reproducible analysis of chemical data.

Author


chromatographR : Chromatographic Data Analysis Toolset

Provides a set of tools for reproducible processing and analysis of HPLC-DAD data or other "simple" chromatographic data (e.g. HPLC-UV, GC-FID) in R. Includes functions for preprocessing, alignment, peak-finding and fitting, peak-table construction, data-visualization, etc.







chromConverter : Chromatographic File Converter

Reads chromatograms from a variety of proprietary formats into R objects. Currently supports conversion of 'Agilent ChemStation', 'Agilent MassHunter', and 'ThermoRaw' files as well as various text-based formats. Provides a simple programmatic interface to file parsers from external libraries, such as Aston , Entab , OpenChrom , and the ThermoRawFileParser .



ShinyChromViewer: Shiny Gadget for Interactive Viewing and Exploration of Chromatograms .



zerenebatchR: Utility for Batch Processing Images in Zerene Stacker .



Maintainer

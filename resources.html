Resources

Some software tools I developed during my PhD to facilitate easier and more reproducible analysis of chemical data.


chromatographR provides a set of tools for reproducible processing and analysis of HPLC-DAD data or other "simple" chromatographic data (e.g. HPLC-UV, GC-FID) in R.






chromConverter provides a set of parsers for reading binary chromatographic data into R.
